Modal

  • HTML
  • JS

Requirements

Opening the modal

  1. Place modal HTML as a direct child of the <body>
  2. Add .u--no-scroll class to the <body>
  3. Add aria-hidden="true" to all siblings of the modal

Inside the modal

  1. Once the modal is open, focus the user on the first focusable element within the modal.
  2. Trap the users focus inside of the modal
  3. Allow the user to close the modal by pressing the escape key

Closing the modal

  1. Add display: none; or remove the modal from the DOM
  2. Remove .u--no-scroll class from the <body>
  3. Remove aria-hidden="true" from all siblings of the modal
  4. Place the users focus on the element that triggered the modal in the first place
